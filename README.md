## 🐍 Snake — 经典贪吃蛇游戏（Python + Pygame）

这是一个使用 **Python 和 Pygame** 实现的经典 *贪吃蛇游戏*，可以在本地电脑上运行和体验。玩家通过键盘控制蛇移动，吃掉食物使蛇变长，同时避免撞到墙壁或自身。

---

## 🎮 游戏玩法

* 使用 **↑ ↓ ← →** 键控制蛇的移动方向
* 吃掉出现在随机位置的食物，蛇身会变长
* 避免撞墙或撞到自己的身体
* 游戏结束时可重启继续挑战

这个游戏实现了贪吃蛇的基本逻辑、碰撞检测和身体增长机制，是学习游戏编程与 Pygame 的一个很好示例。

---

## 🚀 项目结构

```

├── snake.py                # 蛇逻辑（移动、生长、绘制）

```

## 📦 开始玩

### 1. 克隆仓库

```bash
git clone https://github.com/linyi-keke/snake.git
cd snake
```

---

### 2. 安装依赖

本游戏使用 **Pygame**，可以通过 pip 安装：

```bash
pip install pygame
```

如果你使用的是虚拟环境，请确保激活后再安装。

---

### 3. 运行游戏

```bash
python main.py
```

运行之后即可在窗口中体验贪吃蛇游戏。

---

## 🧠 代码亮点

* **面向对象结构**
  基于 `Snake`、`Food` 和 `Game` 类组织逻辑，更易扩展和维护。

* **碰撞检测 & 身体增长**
  吃到食物后身体增长，通过列表记录身体坐标。

* **方向控制逻辑**
  防止反向操作导致蛇直接撞自己。

## ❓ 如何玩更好？

* 保持平稳移动与收尾管理，可以训练自己的节奏感
* 多次吃食物增长后，空间会变得紧张，需要更快反应
* 可以自己加难度设定：让蛇更快、更难控制
